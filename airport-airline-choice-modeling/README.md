# Airport & Airline Choice Modeling

## Project Overview

This project analyzes airport and airline choice behavior using survey data and predictive modeling techniques.

## Objective

To identify key factors influencing passenger decisions and build predictive models.

## Methods

* Logistic regression
* Decision trees
* Feature engineering
* Model evaluation (AUC)

## Tools

Python, pandas, scikit-learn, statsmodels

## Key Results

* Achieved strong predictive performance (AUC up to ~0.89)
* Destination and departure time are the strongest predictors

## Skills

Machine learning, classification, data analysis, business analytics
