
# Causal Impact of DUI Policy (Regression Discontinuity)

## Project Overview

This project evaluates whether harsher DUI penalties at the 0.08 BAC threshold reduce repeat offenses using a regression discontinuity design (RDD).

## Objective

Estimate the causal impact of stricter DUI penalties at BAC = 0.08.

## Methods

* Regression Discontinuity Design (RDD)
* Local linear regression
* Bandwidth sensitivity checks
* Density testing

## Tools

R, fixest, rdrobust, rddensity, ggplot2

## Key Result

Drivers just above the 0.08 threshold are ~1.7–1.8 percentage points less likely to reoffend.

## Skills

Causal inference, econometrics, regression analysis
